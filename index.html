This project is an example of how data mining can be used to obtain useful knowledge taken from trends or correlations of a large dataset. The data mining techniques used for this project were: extract, transform and load (ETL), pearson correlation, boxplots, and chi-squared stats.

The dataset used for this project was taken from the Kaggle competition: NFL 1st and Future - Analytics
